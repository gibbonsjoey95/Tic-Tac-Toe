# Tic Tac Toe

Welcome to Tic Tac Toe, a classic game where two players take turns marking spaces on a 3x3 grid to achieve a winning pattern.

## Overview

Tic Tac Toe is a simple yet entertaining game where players aim to get three of their marks in a row, column, or diagonal. The game ends in a draw if all spaces on the board are filled without a winner.

## Features

- Two-player gameplay: Play against a friend on the same desktop.
- Interactive interface: Click on the grid to mark your move.
- Win detection: The game automatically detects when a player achieves a winning pattern.
- Draw detection: The game declares a draw if all spaces on the board are filled without a winner.

## Installation

To play the Tic Tac Toe game on your desktop:

1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser.
3. Start playing by clicking on the grid to mark your moves.
